# Personal website
My personal website，website address：www.hyram.net

## 目录规划
```
  ├──index.html //主页，链接至photo、video、blog、ext
  └──src
      ├──photo  //主要展示摄影照片等
      ├──video  //主要展示一些自己拍摄制作有意思的视频、或者链接一些其它地方的视频
      └──ext    //主要展示自己动手捣鼓的一些东西的聚合，并带九宫布局存放链接
```
